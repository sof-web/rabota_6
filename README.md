Вариант 1. Жадный алгоритм для вершинного покрытия

Инициализация: создаём пустое покрытие и список всех рёбер.
python

cover = set()
edges = [(0,1), (1,2), ..., (9,0)]

Цикл пока есть непокрытые рёбра: выбираем ребро, добавляем в покрытие обе его вершины (жадный шаг).
python

while edges:
    u, v = edges.pop()  # берём любое ребро
    cover.add(u)
    cover.add(v)

Удаление покрытых рёбер: после добавления вершин удаляем все рёбра, инцидентные этим вершинам.
python

    edges = [e for e in edges if e[0] not in cover and e[1] not in cover]

    Завершение: когда все рёбра покрыты, возвращаем найденное покрытие.

Почему коэффициент ≤ 2?
Алгоритм добавляет 2 вершины на каждое выбранное ребро. Оптимальное решение должно добавить хотя бы 1 вершину на ребро. Значит, размер жадного покрытия ≤ 2 · OPT.
